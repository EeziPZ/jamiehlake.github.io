Code Club ZA

A user-friendly membership website with a fully automated online coding course, featuring custom support for schools and learning centers.

Code Club South Africa was a personal project I embarked on to provide an online coding course for beginners.

The website features a membership system that accepts payments in South African Rands and integrates with the LearnDash LMS for a fully automated experience.

Additionally, I added support for schools and learning centers by creating a front-end interface for them to monitor and interact with their students, including custom redirects to create a personalized experience based on user roles.

Challenges Overcome:

The main challenges in creating this site were developing a membership system that was both user-friendly, fully automated with LearnDash and accepted South African Rands. Another challenge was creating a front-end interface for teachers that was intuitive and easy to use. Despite these challenges, I was able to successfully create a platform that met the needs of my target audience.
